After looking into the SRD, the following Standards need to be abided by for full distribution:

*Races Limitations*
Dwarf - Only Hill Dwarf subrace
Elf - Only High Elf subrace
Halfling - Only Lightfoot subrace
Gnome - Only Rock subrace

*Classes/Subclass Limitations*
Barb - Only Path of the Berserker
Bard - Only College of Lore
Cleric - Only Life Domain
Druid - Only Circle of the Land
Fighter - Only Champion
Monk - Only Open Hand
Paladin - Only Oath of Devotion
Ranger - Only Hunter
Rogue - Only Thief
Sorcerer - Only Draconic Bloodline
Warlock - Only The Fiend
Wizard - Only The School of Evocation

"Background*
Can only use Acolyte Background
